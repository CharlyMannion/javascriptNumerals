# Roman Numerals

This is a program that takes a number between 1 and 1000 and returns the corresponding roman numeral.

## Brief guide to roman numerals:

```
1 = I
5 = V
10 = X
50 = L
100 = C
500 = D
1000 = M
```

### How to use this programme

To run the programme in the console:

```
roman = new Roman();
```
